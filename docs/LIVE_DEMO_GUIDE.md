# 🖥️ Live Project Demonstration & Viva Command Guide

This guide is designed to help you run a live demonstration of your **Environmental Sound Classification (ESC)** project in front of your professor ("Sir"). It lists every command you need to execute, shows the expected terminal output, and tells you exactly what to say to explain the results.

---

## 🚀 Step-by-Step Live Demo Simulation

### Step 1: Activate Virtual Environment
Open your terminal, navigate to the `ESC_Project` directory, and activate the virtual environment:
```bash
source .venv/bin/activate
```
* **Spoken Explanation**: 
  > *"Sir, first we activate our isolated virtual Python environment which contains all locked production dependencies like PyTorch 2.6+, Transformers 5.13+, and Torchaudio, ensuring reproducibility."*

### Step 4: Show the Confusion Matrix Plot
Open the generated confusion matrix image in your local system viewer:
```bash
# On Mac:
open outputs/confusion_matrix.png
```

#### What to Highlight on the Screen:
Point to the dark diagonal cells (correct predictions) and the off-diagonal points showing minor confusions.

* **Spoken Explanation**:
  > *"Sir, this is the 50x50 confusion matrix generated by our evaluation script. The strong diagonal line shows that the model classifies the majority of the 50 categories correctly. 
  > 
  > We observed minor acoustic confusions. For instance, `wind` is confused with `train` 4 times, and `mouse_click` is confused with `keyboard_typing` 4 times. This makes acoustic sense: wind and train passing in the distance both exhibit flat, low-frequency pink noise structures. Mouse clicks and keyboard typing are both composed of high-frequency click transients with rapid decay envelopes."*
